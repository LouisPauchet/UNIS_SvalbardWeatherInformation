What is SWI ?

Svalbard Weather Information (or SWI) is a web app initially to display real time data from the weather station accross the archipelago as well as regrouping the differents map layers from multiple provider as the Norwegian Polar Institute (NP), the Norwegian Water Resources and Energy Directorate (NVE) and the Norwegian Meteorological Institute (Met Norway)

Where is it from

SWI was first developped as a fun project base on opensource development. The first demo hosted on a crappy server had quite an intereste as the second month, almost 200 persons have been using the app.
