About SWI

Svalbard Weather Information (SWI) is a project designed to provide online access to GIS data essential for planning trips in Svalbard. This application consolidates various layers, including regulations, topography, sea-ice conditions, and real-time weather data from UNIS and Met Norway weather stations. Detailed credit for these data sources is provided below. It serves as a tool to aid in trip planning and should be used solely for informational purposes, not for navigation.

Origins of SWI

SWI was initially developed as a fun, open-source project. The first demo, hosted on a modest server, garnered significant interest; within the second month, nearly 200 users had accessed the app.
